SpendWise – Smart Expense Manager (Java Console App)

SpendWise is a simple and efficient Java-based console application designed to help users track and manage their daily expenses. 
This project demonstrates core Java concepts such as Object-Oriented Programming (OOP), ArrayLists, loops, conditionals, and user input handling using Scanner.

 Features

- Add new expenses (Amount, Category, Date)
- View all recorded expenses
- Search expenses by category
- Delete expense by ID
- Calculate total expenses
- Auto-generated Expense ID system


Technologies Used

- Java (Core Java)
- IntelliJ IDEA
- Git & GitHub


Concepts Implemented

- Object-Oriented Programming (Classes & Objects)
- Encapsulation (Getters)
- ArrayList for dynamic data storage
- Loops & Conditional Statements
- Method-based modular coding
- Basic CRUD operations

How to Run

1. Open project in IntelliJ IDEA
2. Run `SpendWiseApp.java`
3. Use the menu options in console
